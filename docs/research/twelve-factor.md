# Twelve-Factor Alignment

## Sources
- The Twelve-Factor App methodology for building SaaS services with portability, continuous deployment, and scalability. [Link](https://12factor.net/)

## Notes
- Use declarative automation, isolate dependencies, and store config in the environment to keep deployments portable. [Link](https://12factor.net/)
- Treat backing services as attached resources, run stateless processes, and separate build, release, and run stages to enable scaling. [Link](https://12factor.net/)
